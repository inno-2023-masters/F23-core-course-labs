# Lab 1 – Web application

Simple Jaguar web app that displays current time in Moscow

## Structure

```
.
├── app
│   ├── bin
│   └── lib
│   │   ├── routes
│   │   ├── models
│   └── static
```

## Installation & Building

```bash
dart pub get
```

## Running

```bash
dart run
```

## Linters

```bash
dart format .
```
