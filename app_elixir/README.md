## Getting Started

Easy to start with the following guide:

### Prerequisites

- `asdf`

### Installation

```bash
$ asdf install
$ mix deps.get
```

### Usage

```bash
$ cd devops_web
$ mix phx.server
```

Access http://0.0.0.0:4000/time to get MSK time
