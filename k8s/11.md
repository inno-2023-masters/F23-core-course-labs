# 11

#### Get Secrets
```
$ kubectl get secrets
NAME           TYPE     DATA   AGE
db-user-pass   Opaque   2      6s
```

```
$ kubectl get secret db-user-pass -o jsonpath='{.data}'
{"password":"bXlTdHJvbmdQYXNzd29yZA==","username":"YWRtaW4="}
```

```
$ kubectl get secret db-user-pass -o jsonpath='{.data.password}' | base64 --decode
myStrongPassword
```
