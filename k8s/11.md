## Kubernetes Secrets

Create
```
kubectl create secret generic my-age --from-literal=myrealage=52
```

Decode
```
kubectl get secret my-age -o jsonpath='{.data.myrealage}' | base64 --decode >> 11.md
```

Decoded value = 52
