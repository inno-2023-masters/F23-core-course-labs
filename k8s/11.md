### Creation via `kubectl`

```bash
kubectl create secret generic secret-example --from-file=secret=./secret.txt
secret/secret-example created
```
```bash
kubectl get secret secret-example
NAME             TYPE     DATA   AGE
secret-example   Opaque   1      2s
```

### Manage secrets using Helm
After the GPG key and `app-python-helm/secrers.yaml` generation
```bash
cd app-python-helm
```
```bash
helm secrets install . -f secrets.yaml --generate-name
[helm-secrets] Decrypt: secrets.yaml
NAME: chart-1699983701
LAST DEPLOYED: Tue Nov 14 20:41:41 2023

  export POD_NAME=$(kubectl get pods --namespace default -l "app.kubernetes.io/name=app-python-helm,app.kubernetes.io/instance=chart-1699983701" -o jsonpath="{.items[0].metadata.name}")
  export CONTAINER_PORT=$(kubectl get pod --namespace default $POD_NAME -o jsonpath="{.spec.containers[0].ports[0].containerPort}")
  echo "Visit http://127.0.0.1:8080 to use your application"
```
